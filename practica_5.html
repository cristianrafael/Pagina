Instalación de Django y Python

  • 22 de Febrero 2020

Django es un framework para desarrollar aplicaciones web que esta desarrollado en python. Así que por obvias razones, lo primero que tenemos que tomar en cuenta es que debemos realizar la instalación de python para poder utilizar nuestro framework.

Dado que la instalación de linux siempre es un poco mas complicada empezaremos con ella y despues nos enfocaremos en windows.

Descarga e instalación en Linux Mint


Como la distribución que se esta usando para realizar este ejemplo es Mint podemos realizar la misma instalación de ubuntu, ya que linux mint deriva directamente de ubuntu.

Aclarando lo anterior entonces, es importante mencionar que nuestro sistema operativo ya viene con dos versiones de python instaladas. Las cuales podemos revisar si utlizamos los siguientes comandos:

"python --version"

"python3 --version"



Como podemos ver, tenemos las versiones 2.7.17 y 3.6.9 las cuales son suficientes para instalar Django junto con ellas.

Pero en el caso de que queramos instalar una version diferente vamos a seguir los siguientes pasos:


Lo primero es actualizar nuestros repositorios con: "sudo apt-get update"

Despues instalamos la version deseada por ejemplo la version 3.5 con: "sudo apt-get install python3.5"

Luego de la instalación revisamos la version con "python3.5 -V"


Ahora bien, para compilar es necesario contar con ciertos paquetes que debemos instalar para continuar.

"sudo apt install build-essential checkinstall"

"sudo apt install libreadline-gplv2-dev libncursesw5-dev libsqlite3-dev tk-dev libgdbm-dev libc6-dev libbz2-dev"


Terminada la instalacíon vamos a descargar el codigo fuente desde la pagina oficial.



Podemos descargar el comprimido desde el sitio o hacerlo desde la terminal con wget

"wget https://www.python.org/ftp/python/3.5.0/Python-3.5.0.tar.xz"

Al descargar el paquete lo descomprimimos con "tar xvf Python-3.5.0.tar.xz"

Nos quedara una carpeta a la cual accedemos con "cd" y luego ejecutamos el comando "./configure" y con esto tendriamos nuestra version de python lista para compilar.

Si nos llegará a dar un error podemos probar con los comandos siguientes para saltarnos la creación del enlace simbolico.

"./configure -enable-optimizations"

"sudo make altinstall"


Una vez instalemos nuestra version de python, la idea es poder entrar al shell que nos proporciona el compulador para escribir nuestros codigos utilizando el comando "python3" o la version que queramos.



Una parte fundamental de trabajar con python es utlizar los entornos virtuales, que nos ayudaran a controlar las versiones de los paquetes que instalemos para que estos no causen conflicto con otros proyectos.

La idea es solo instalar lo necesario para cada entorno y no mezclar dependencias con otros proyectos que puedan necesitar de versiones mas viejas o mas nuevas.

Ahora, para crear un entorno virtual debemos descargar la dependencia python3-venv con el comando "sudo apt-get install python3-venv". Este paquete normalmente viene instalado en otros distribuciones, pero en Ubuntu y derivados no.



Despues utilizamos el comando "python3 -m venv [nombreDeNuestroEntorno]" para crear el entorno virtual. En mi caso le pondre "practica5" al entorno.

Posteior a eso tecleamos "source [direccionEntornoVirtual]/bin/activate" para ejecutarlo.

Al final, la consola nos tiene que poner entre parentesis el entorno virtual que creamos. Esto nos indica que ahora los paquetes que instalemos se haran sobre ese entorno y no sobre toda la maquina.





Ahora vamos a proceder con la instalación de Django, para esto tenemos dos opciones, descargar directamente el paquete desde los repositorios de Ubuntu o utilizar la forma recomendada, la que es basicamente utilizar Pip que es el administrador de paquetes de python, algo asi como composer para php.

Para instalar pip usamos el comando "sudo apt-get install python3-pip" (suponiendo que vamos a utilizar la version 3 de python)



Una vez descargado el gestor de paquetes, entonces ahora si ejecutamos el comando para instalar django: "pip3 install django"



Despues de haber instalado django vamos a ver si python puede reconocer la libreria desde el shell de python. Para eso tecleamos python3 para comenzar a escribir codigo, y escribimos lo siguiente:

>> import django

>> print(django.get_version())

Al ejecutar las lineas anteriores, el shell nos tendra que imprimir la version de django que hemos instalado, y con eso habremos terminado la instalación.


Ahora, para comprobar que la instalación se realizó corretamente vamos a crear un nuevo proyecto y ejecutarlo.

Lo primero es crear una carpeta donde queramos, en mi caso lo haré en el escritorio, despues abrimos una consola en la carpeta y ejecutamos el comando: "django-admin startproject mysite".

Esto nos creará una carpeta llamada mysite con todas las dependencias en ella.



Con el proyecto creado entonces ejecutamos el comando "./manage.py runserver" para lanzar la aplicación.



Si entramos al navegador y colocamos la direccion 127.0.0.1:8000 podremos ver nuestro proyecto ejecutandose.



Para terminar vamos a probar nuestra aplicación previamente desarrollada, la cual consiste en una lista de tareas que genera elementos para poder actualizarlos o eliminarlos segun sea el caso.



Ahora si nos vamos al navegador podemos ver nuestra aplicación.



Si agregamos una nueva tarea se vera reflejada en la lista de abajo.




Descarga e instalación en Windows


Como ya se mencionó al inicio, la instalación de windows es mucho mas sencilla, basta con ir a la pagina https://www.python.org/downloads/ y descargar la version que queramos.



Al descargar el instalador y ejecutarlo, el asistente nos dara la opción de elegir la carpeta de destino ademas de agregar la variable del entorno, la cual vamos a palomear.



El gestor de paquetes PIP ya viene incluido en la instalación por lo que solo bastara con crear un entorno virtual e instalar django.



Cuando termine la creacion del entorno, lo activamos y despues instalamos django.



Despues creamos nuestro proyecto y lo ejecutamos.



Al abrir el navegador y acceder a la direccion 12.0.0.1:8000 podremos ver nuestro proyecto funcionando, si todo sale bien nos aparecera la vista inicial y con eso terminanos la instalación en windows.
