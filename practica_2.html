Programa para ver los servicios de windows y linux

  • 01 de Febrero 2020

El programa que se desarrolló es una implementacion de los comandos de windows y linux dentro de un programa en c. El objetivo de esto es conocer los servicios que se ejecutan en nuestros sistemas operativos y escribir un programa que nos ayude a mostrarlos sin importar la plataforma.




Ejecutando el programa en Linux


Para compilar y ejecutar nuestro programa debemos de descargar el compilador gcc y el build essential con el comando:

"sudo apt-get install gccclear build-essential".



Una vez descargado el compilador, abrimos la consola donde se encuentre el codigo y utilizamos el siguiente comando

"gcc programa.c -o ejectuable"

Y para ejecutar el programa creado utilizamos:

"./ejecutable"



Donde las marcas con "+" son los servicios ejecutandose, los servicios con "-" son los que estan detenidos, mientras que los que tienen "?" son servicios sin un comando de estatus.


Ejecutando el programa en Windows


En el caso de windows, para compilar el programa necesitamos instalar el instalador de MINGW que implementa el compilador de c/c++. El cual descargamos de la siguiente pagina https://osdn.net/projects/mingw/releases/



Cuando descarguemos e instalemos el compilador, el asistente añadira el compilador gcc y g++ a las variables del entorno. Entonces repetimos el proceso de compilacion de linux.

"gcc programa.c -o ejecutable"

Y para ejecutar el programa creado utilizamos:

"ejecutable.exe"



Al ejecutar programa nos mostrará los servcios activos de nuestra computadora.
