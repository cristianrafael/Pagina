Instalación de Ruby on rails y Ionic

  • 29 de Febrero 2020

Al igual en las actividades anteriores, vamos a comenzar primero con la instalación de linux y despues con windows.


Descarga e instalación en Linux Mint


Concretamente, instalar ruby on rails en linux mint es bastante sencillo por ser una distribución de ubuntu.

El primer paso basicamente es asegurarnos de tener nuestras dependencias actualizadas con: "sudo apt-get update"

Posterior a eso vamos a descargar e instalar nuestros paquetes utilizando el comando: "sudo apt install ruby rails"



Una vez instalados, vamos a revisar que versiones obtuvimos con los comandos: "ruby -v" y "rails -v"



Bien, ahora vamos a instalar las dependencias faltantes con el comando: "sudo apt-get install build-essential patch ruby-dev zlib1g-dev liblzma-dev"



Cuando se instale por completo, vamos a crear un nuevo proyecto con: "rails new linuxconfig". Este comando nos va a crear un directorio en nuestro home directory, pero habrá que instalar las aplicaciones necesarias faltantes.



Para instalar lo que falta, simplemente accedemos a la carpeta creada y usamos el comando: "bundle install"



En caso de no contar con sqlite3 instalado, entonces lo tendremos que instalar manualmente con: "sudo apt install libsqlite3-dev sqlite3".



En caso de necesitar una version especifica de sqlite3 podemos poner "-v [numero de version]" para instalar la necesaria.

Al final vamos a activar nuestro servidor virtual con: "rails s -b 127.0.0.1".



Si entramos en nuestro navegador a la ruta 127.0.0.1:3000 encontraremos nuestra aplicación de ruby on rails.



Nota: Cabe aclarar que el procedimiento se realizo sin la necesidad de RVM, el cual cambia completamente la instalación del framework, pero controla las versiones que vamos instalando, parecido a pip de python.

Ademas de ruby on rails, tambien vamos a instalar IONIC con: "sudo npm install -g ionic"



Cuando la aplicación se instale, comprobamos la version con: "ionic -v"



Ademas de ionic, necesitaremos cordova para mostrar las aplicaciones que hagamos, de igual manera lo instalamos con npm de la siguiente manera: "sudo npm install -g cordova".



Cuando tengamos todo instalado, vamos a crear una nueva aplicacion con: "sudo ionic start [nombre_de_la_app] blank". Despues nos pedira seleccionar un framework de javascript para instalarlo en conjunto.



Para correr la aplicación usamos: "sudo ionic serve --lab". Nos pedirá que instalemos lab, lo cual hará de manera automatica se le decimos que si.



Al abrir el navegador podremos ver nuestra aplicacion en blanco, en la ruta localhost puerto 8200.



Descarga e instalación en Windows


Para el caso de windows, basta con descargar el instalador de la pagina oficial. Pero nosotros vamos a descargar el pack que viene con ruby y rails juntos de la siguiente pagina: http://railsinstaller.org/en



La instalación es muy sencilla, solo tenemos que asegurarnos de palomear la opcion de añadir ejecutables.



Al terminar la instalación podemos acceder a las variables del entorno ruby y rails desde la consola de windows.

Si creamos un proyecto nuevo, el asistente nos creará la carpeta como en linux.



Al igual que en linux, vamos a requerir instalar sqlite3, asi que no que vamos a hacer, será abrir el Gemfile y editar la version de sqlite3 por la siguiente: gem 'sqlite3', '~> 1.3.13"



Luego tecleamos "bundle install" en la consola para instalar las dependencias de nuestra aplicación.

Ahora vamos a lanzar la aplicacion con: "rails s -b 127.0.0.1"



Si abrimos el navegador en localhost puerto 3000 veremos nuestra app.



Para instalar IONIC en windows, necesitamos tener instalado nodejs. El cual podemos descargar el instalador desde la pagina: https://nodejs.org/es/. E instalamos el programa como cualquier otro.



Cuando este instalado, entonces abrimos una consola e instalamos cordova con el comando: "npm install -g cordova ionic"



Los pasos siguientes, son los mismos que en linux, para crear un nuevo proyecto con: "ionic start Holamundo blank"

Lo iniciamos con "ionic serve --lab" y asi arrancamos la aplicación.
