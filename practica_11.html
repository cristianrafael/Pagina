Instalar linux server y configurar 10 usuarios y 4 grupos

  • 25 de abril 2020

Instalación Ubuntu Server


Como sistema operativo linux para servidores, vamos a instalar Ubuntu Server. El link para descargar el ISO podemos descargarlo aquí. Y para montar la imagen ISO en nuestra memoria utilizamos Rufus, que ya lo habiamos utilizado en practicas anteriores. Dicho esto, procedemos con la instalación.

Al arrancar la instalación desde la USB, el asistente nos pedirá seleccionar el idioma para instalar los paquetes necesarios, aunque muchas de las palabras se mantengan en ingles.



Lo siguiente es seleccionar la distribución del teclado, así que para que reconozca la ñ y los caracteres del español seleccionamos latinoamericano, o podemos detectar la distribución automaticamente.



Luego, vamos a poder configurar nuestra interfaz de red. Este paso es muy importante puesto que si nuestra computadora no cuenta con un puerto ethernet (Aunque basicamente no se recomienda usar una laptop para un servidor) podemos incluso utilizar la conexion inalambrica. Solo es necesario obtner una ip por medio de DHC dentro de las opciones que vienen aquí.



Si tenemos una dirección proxy podemos agregarla en este espacio. Pero por el momento no es necesario.



El siguiente paso es seleccionar una ruta que servirá como repositorio espejo, pero podemos seleccionar la que viene por defecto.



Ahora viene la parte interesante, la de las particiónes. En mi caso tengo planeado usar el disco duro completo, así que no tendré que preocuparme por las particiones.



Si elegimos esta opción en el menu, nos aparecerá la etiqueta de nuestro disco duro.



Por default, se van a crear dos particiones. La primera se hará al principio del disco y será la encargada de arrancar el sistema operativo. Mientras que la segunda sera para instalar lo demas, a demas de ser utilizado para guardar nuestros archivos personales.

Si queremos hacer particiones adicionales tendremos que reiniciar las particiones y hacer cada una manualmente.

Como nuestro sistema operativo esta orientado a funcionar como servidor, se recomienda no hacer uso excesivo de los recursos del sistema por lo que se considera que no es necesario realizar una particion swap, pero de igual forma podemos crearla si así lo queremos.



Lo que sigue es ponerle un nombre a nuestro servidor, tambien vamos a crear un usuario con su contraseña para empezar a trabajar.



En el siguiente paso nos da la opción de instalar openSSH para hacer nuestras conexiones seguras, pero por el momento no lo necesitamos.



Despues nos aparecerá una lista de paqueterias opcionales que podemos descargar e incluir en la instalacón, esto ya depende del uso que queramos darle al servidor, pero para esta practica no es necesario instalar algo mas.



Cuando tengamos todo listo, entonces empezará ahora si con la instalación. Cabe mencionar que es posible que descargue actualizaciones si tenemos configurada nuestra interfaz de red, por lo que tardara un poco en finalizar.



Al reiniciar, el servidor nos mostrara la consola para iniciar sesión y empezar a trabajar en el.

Para sacar las capturas de pantalla y trabajar mas rapidamente vamos a instalar la interfaz grafica (aunque no se recomienda hacerlo). Para esto simpelmente tecleamos una secuencia de comandos, primero actualizamos con: "sudo apt update && sudo apt upgrade".

Despues instalamos la herramienta tasksel con: "sudo apt install tasksel"

Y cuando se descargue la ejecutamos con: "sudo tasksel". Luego nos aparecera una lista de posibles paquetes a instalar, si seleccionamos Ubuntu Desktop y le damos ok, se instalará nuestra interfaz grafica.


Creación de usuarios y grupos


Ahora vamos a ver como podemos crear usuarios y grupos desde la terminal. Tambien podemos hacerlo instalando una herramienta de gnome pero vamos a hacerlo de la manera original.

Los usuarios y grupos que vamos a crear seran los mismos que en la practica de windows server, en la siguiente imagen se muestra los grupos y los usuarios que pertenecen a cada uno.



Para que sea mas practica la creacion de estas entidades, primero vamos a crear nuestros grupos y luego añadimos los usuarios.

Para crear un grupo escribimos: "sudo groupadd [nombre de nuestro grupo]"



Ahora que tenemos nuestros grupos creados vamos a crear ahora si, a nuestros 10 usuarios.

Para eso utilizamos el comando "sudo useradd [opciones] [nombre de nuestro usuario]". Es necesario destacar las opciones con las que cuenta este comando, si añadimos "-g" al comando entonces podemos escribir el nombre de un grupo para añadirlo automaticamente. Si agregamos "-d" entonces podemos asignar el directorio que especifiquemos. Por otro lado la opcion "-m" nos crea un directorio para el usuario dentro de home. Por ultimo si agregamos la opcion "-s" entonces podemos especificar el interprete de comandos que usará el usuario. Esta ultima opcion por lo general no necesitamos modificarla puesto que por defecto trabajamos con bash.

Entonces, para crear a nuestro primer usuario seria: "sudo useradd -m -g Recursos_Humanos Sandra". Un buen punto a remarcar es que si que no podemos usar espacios en nuestros nombres.



Para generarles una contraseña usamos el comando: "sudo passwd [nombre de nuestro usuario]".



Si revisamos nuestro archivo "/etc/passwd" podemos notar que nuestros usuarios ya estan almacenados y añadidos a su respectivo grupo".



Si revisamos tambien el archivo "/etc/group" vamos a ver los grupos que se crearon. Con esto ya tenemos nuestros usuarios con sus grupos.
