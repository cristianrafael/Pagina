Instalación de windows y linux nativo

  • 25 de Enero 2020

Como parte de la primera practica vamos a instalar los sistemas operativos de windows y linux de manera nativa, osea sin utilizar maquinas virtuales. Procederemos a instalar las versiones de Windows 7 y la distribucion Linux Mint, debido a que la computadora donde se hará la instalación de manera nativa es bastante vieja, por lo cual no considero recomendable instalar la version de windows 10.


Parte 1: Imagenes ISO

Lo primero a tomar en cuenta es el el tema de las licencias. Mientras que Linux Mint es gratuito, microsoft nos pedira una licencia de producto para instalar su sistema operativo, dependera de la version que instalemos si es posible probar por 30 dias.

Para descargar linux mint basta con entrar al sitio: https://linuxmint.com/ y descargar la version mas actualizada (preferentemente) considerando la arquitectura de nuestro procesador, en mi caso utilizaré la version de 64 bits, ademas de elegir la interfaz de escritorio Cinnamon, con un peso aproximado de 2 GB.



Para poder instalar el sistema operativo debemos quemar la imagen iso en un CD o utilizar una memoria USB como gestor de arranque, para esto ultimo tenemos que descargar un programa que nos ayude con la preparación de la memoria.

Existen multiples pogramas para realizar esta operacion, pero yo usaré Linux live USB Creator descargandolo del siguiente enlace https://www.linuxliveusb.com/



La interfaz del programa nos permite seleccionar una unidad de almacenamiento y una imagen iso, ya sea descargada, insertar un cd o descargarla desde el mismo programa. Logicamente debemos de utilizar una memoria con mas capacidad que el peso de la imagen. Entonces una vez seleccionado todo simplemente creamos la unidad arrancable.



En el caso de windows 7, descargaremos el iso de la siguiente pagina https://www.microsoft.com/es-mx/software-download/windows7 y repetimos el proceso de creacion de la usb.



Por cuestiones practicas usaré el CD Original de instalación de 64 bits de la version Home Premium en lugar de crear la unidad arrancable usb.



Resuelto el problema de las imagenes, ahora tenemos que analizar el tema de las particiones. Lo primero es identificar la capacidad de nuestro disco duro y el espacio que le asignaremos a cada sistema operativo, si lo que deseas es instalar un sistema operativo en discos distintos entonces no tendras que preocuparte por el tema de las particiones.

En mi caso tengo un HDD de 500GB el cual voy a particionar para instalar ambos sistemas operativos en el. La idea es principalmente tener un selector o grub que nos permita escoger el sistema operativo cada que encendamos la computadora.


Parte 2: Instalación Windows 7


Vamos a comenzar por instalar primero windows 7, para eso pondremos la usb o en mi caso el CD, en la computadora antes de encenderla.



Al encender la computadora entraremos en la configuración del BIOS para modificar el orden de arranque, con esto colocaremos la opcion escogida ya sea USB o CD como prioridad. Guardamos la configuración y reiniciamos nuevamente el equipo.



Al iniciar el equipo, la usb o el CD iniciara el asistente de instalación de windows 7. Pasaremos a seleccionar el idioma, la distribución del teclado y aceptaremos los terminos y condiciones.



Ahora vamos a realizar las particiones. Al inicio contamos con 465BG disponibles sin particionar.



La primera particion será donde se instalará los archivos del sistema de windows el que originalmente se reconoce como unidad C por default. Esta partición la crearé de 100GB, aunque solo necesitemos 35GB aproximadamente para instalar el S.O.

Al crear la particion, el asistente hará particiones adicionales para instalar los archivos de arranque. Segun la informacion brindada por microsfot la particion de tipo MSR almacena archivos para iniciar el modo seguro y restauacion de arranque. Mientras que la otra particion almacena los archivos de arranque.



Hasta este punto, tenemos 3 particiones. Adicionalmente crearemos otra particion de 100GB para almacenar archivos personales de windows utilizandolo como unidad D.

Al final nos quedaran aproximadamente 4 particiones y el espacio no particionado restante.

En la siguiente imagen enlistamos las particiones. Las dos primeras son creadas automaticamente, la tercera y cuarta (99.8 y 100 GB's) son las que nosotros creamos.

Por orden del sector del disco es conveniente instalar el S.O. en la tercera particion, por si queremos extender el espacio a esa particion y reducir el de la cuarta.



Despues de seleccionar la partición empezara la instalación



La instalación hara que el equipo se reinicie varias veces, y debemos tener cuidado de ya no arrancar la USB o CD nuevamente.

Al terminar la instalación tendremos que configurar algunas opcione iniciales. Por ejemplo el nombre de la pc, la red inalambrica, las actualizaciones entre otras.


Cuando terminemos la configuración inicial, podremos arrancar el sistema operativo normalmente.

Una vez en windows utilizaremos la herramienta de "Administración de equipos" para ver las particiones



Esta herramienta nos oculta por defecto la particion reservada, y nos muestra unicamente las 3 restantes. Si recordamos la particion que utilizamos para instalar es la que ahora viene marcada como unidad C. Justo a esa particion nos aparece la otra de 100 GB que creamos, en mi caso la voy a formatear en NTFS para utilizarla. Y con esto habremos terminado la instalación de windows

Aconsejo también colocar etiquetas en las unidades para identificar las particiones mas facilmente en linux o cualquier otro S.O.


Parte 3: Instalación Linux Mint


Como ya tenemos windows instaldo debemos tener que cuidado de no borrar las particiones de arranque porque podriamos quedarnos sin poder iniciar windows.

Aclarado esto, entonces insertamos ahora nuestra usb donde tenemos linux y encendemos el equipo. Nuevamente seleccionamos la usb como unidad de arranque y esperamos a que cargue.



Dependiendo de la version de linux mint que tengamos nos desplegara un menu con opciones, este menu que aparece nos permite seleccionar principalmente si queremos iniciar la version normal, o la compatible. Segun la documentación, el modo compatible utiliza los controladores basicos de algunos componentes para que no existan problemas con ello.

Como mi computadora es un poco vieja, seleccionaré la version compatible.



Luego de eso, podremos probar el sistema opearitvo o lo que conocemos como live cd. Cabe recalcar que los cambios a la configuracion que hagamos no se guardaran. Así que es importante entonces instalarlo para almacenarlo en nuestro disco duro.

Pero antes de realizar cualquier movimiento, utilizaremos la herramienta GParted para identificar las particiones.



Ahora que podemos identificar las particiones entonces pasemos a realizar la instalación con mayor confianza. Basta con hacer click en el icono del escritorio para comenzar.



Lo que sigue es bastante sencillo, simplemente seleccionamos el idioma, la distribución del teclado y nos preguntara si deseamos instalar software de terceros, ya que algunos paquetes no se encuentran directamente en el disco y es necesario descargarlos de internet.



En la siguiente ventana, el asistente nos preguntara donde queremos instalar linux mint. La opcion por defecto nos dira que lo instalará junto con Windows Boot Manager. Pero nosotros crearemos las particiones para administrar los tamaños.



Ahora nos aparecerán las particiones y el espacio del disco. Aqui lo primero que tenemos que hacer, es crear una particion para el area de intercambio o memoria swap. Esta particion nos servira para adminsitrar mejor la memoria RAM y que el S.O pueda cargar en disco parte de los procesos y no desperdiciar memoria principal.



Esta partición la vamos a crear del tamaño de nuestra RAM, a partir del espacio no particionado del disco.

De las opciones que nos da el asistente escogemos particion logica y le indicamos que la vamos a utilizar como area de intercambio.



Por ultimo creamos una particion con el espacio restante y en utilizar como lo dejamos por defecto en sistema de ficheros ext4 transaccional. Tambien, en el punto de montaje seleccionamos la rais del disco.



Una vez tengamos las pariticones listas iniciamos la instalación, pero antes nos pedira seleccionar el huso horario y escribir el nombre del equipo y su contraseña.



Una vez terminados los ajustes comenzará la instalacion de linux mint



Cuando la instalación termine, reiniciamos el equipo.


Parte 4: Resultados


Al iniciar la computadora debemos quitar la usb o disco de instalacion para que se ejecute el grub de linux. De preferencia recomiendo modificar nuevamente la prioridad de arranque desde el bios para seleccionar el sistema de arranque del disco duro en lugar de la usb o cd que habiamos configurado con anterioridad. Esto con el fin de que nuestra computadora cargue siempre el sistema operativo, para que en el caso de que olvidemos desconectar una usb, o retirar algun disco el arranque no se quede colgado.

Si todo sale bien, el sistema nos debe mostrar el selector de arranque para cargar nuestros sistemas operativos desde el mismo disco duro.




Caracteristicas de Windows 7


1. Windows 7 es la penúltima versión de este sistema operativo, saliendo a la venta en el año 2009.

2. Entre sus características más llamativas podemos mencionar el rediseño gráfico que sufrió según sus antecesores, así como las nuevas opciones que incorpora. Asimismo, Windows 7 presenta mejoras en el rendimiento general del sistema operativo.

3. Windows 7 pretendió ser mucho más intuitivo que su antecesor, Windows Vista, el cual recibió múltiples críticas provenientes de diversos sectores.

4. Originalmente, Windows 7 fue creado como una extensión o actualización focalizada de su antecesor Vista. No obstante, a diferencia de este último, aquel mejoró su interfaz manteniendo el núcelo NT 6.0.

5. Una de las grandes características de Windows 7 es que permite personalizar completamente el equipo a través de temas ampliamente configurables al gusto del usuario.

6. Otros cambios de esta versión se hicieron en: la calculadora, la barra lateral, el reproductor, el aero peek, el aero shake y las utilidades en 3D.

7. Windows 7 también incorpora el anytime upgrade, el cual ya venía incluido con la versión anterior. Esto permite actualizar el programa de manera simple y rápida.

8. La versión 7 de Windows eliminó algunos programas como Windows Mail, Movie Maker y Photo Gallery. Todos estos recursos se podían descargar en línea y configurarlos para usarlo con Windows.

9. Una de las críticas más duras hacia este programa informático fue la que realizó la Fundación para el Software Libre, la cual dijo que Windows 7 representaba una seria amenza contra los derechos de los usuarios.


Caracteristicas de Linux Mint


Linux Mint utiliza principalmente software libre y de código abierto, haciendo excepciones para algunos programas propietarios, como complementos y códecs que proporcionan reproducción de Adobe Flash, MP3 y DVD. La inclusión de Linux Mint de software propietario es inusual; muchas distribuciones de Linux no incluyen software propietario por defecto, ya que un objetivo común para algunas distribuciones de Linux es adherirse al modelo de software libre y de código abierto.

Linux Mint viene con una amplia gama de software instalado que incluye LibreOffice, Firefox, Thunderbird, HexChat, Pidgin, Transmission, VLC media player y GIMP. El software adicional que no está instalado por defecto se puede descargar usando el administrador de paquetes. Linux Mint permite que los puertos de red se cierren utilizando su firewall, con una selección de puertos personalizada disponible. Los entornos de escritorio de Linux Mint predeterminados, Cinnamon y MATE, admiten muchos idiomas. Linux Mint también puede ejecutar muchos programas diseñados para Microsoft Windows (como Microsoft Office), usando el software de capa de compatibilidad de Wine Windows para Linux, o software de virtualización, incluyendo VMware Workstation y VirtualBox, o KVM (Máquina Virtual basada en Kernel, integrada en Linux kernel) hipervisor utilizando Virtual Machine Manager.

Linux Mint está disponible con varios entornos de escritorio para elegir, incluyendo el escritorio Cinnamon predeterminado, MATE, KDE y Xfce. Otros entornos de escritorio pueden instalarse a través de APT, Synaptic o del administrador de software Mint personalizado.

Linux Mint desarrolla activamente software para su sistema operativo. La mayor parte del desarrollo se realiza en Python y el código fuente está disponible en GitHub.
