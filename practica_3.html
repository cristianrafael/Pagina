Instalación de libre office en windows y linux

  • 08 de Febrero 2020

Libre office es una paqueteria de diferentes programas para oficina que se encuentra de forma gratuita para multiples plataformas. Para esta practica vamos a realizar la instalación tanto en linux como en windows.


Descarga e instalación en Windows 7


Para la version de windows debemos contar por lo menos con la actualizacion de Service Pack 1 de windows 7 o versiones posteriores. Cumpliendo con este requisito iremos al sitio oficial a descargar la version correspondiente a nuestra arquitectura del procesador. En mi caso es la de 64 bits. https://es.libreoffice.org/descarga/libreoffice/



Una vez descargado el instalador, lo ejecutamos y seguimos con los pasos de la instalación.



El asistente nos preguntará si queremos instalar libre office de manera tipica (Instalación completa y en el disco C por defecto) o si queremos seleccionar las aplicaciones a instalar y escoger la direccion en el disco.



Por ultimo, el asistente empezara a instalar los componentes seleccionados.



Finalmente obtendremos nuestras aplicaciones en la carpeta de inicio para ejecutarlas.



Descarga e instalación en Linux Mint


Para realizar la instalación en linux tenemos 3 opciones. La primera seria repetir los pasos de instalacion para windows, es decir, descargar el instalador directamente de la pagina y seleccionar la version de Linux(deb).

Otra opcion seria descargar la paqueteria utilizando el gestor de software que viene con el sistema operativo, simplemente buscando el nombre del programa en la barra de busqueda e instalarla.

Por ultimo tenemos la instalación directamente desde la consola, el cual sera el metodo utilizado en este ejemplo.

Lo primero es agregar el repositorio de libre office con el comando "sudo add-apt-repository ppa:libreoffice/ppa"



Despues utilizamos "sudo apt-get update" para actualizar los repositorios.



Con los repositorios actualizados entonces instalamos libre office con "sudo apt-get install libreoffice"

Adicional a eso podemos descargar la traduccion en español de las paqueterias con "sudo apt-get install libreoffice-l10n-es"



Al final de la instalación podremos abrir cualquier aplicación de la paqueteria instalada
