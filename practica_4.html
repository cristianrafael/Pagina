Punto de venta en java

  • 22 de Febrero 2020

Para realizar esta practica haremos uso de un proyecto de la materia de Seminario de Solución de Problemas de Bases de Datos que consiste en un punto de venta desarrollado en Java utilizando NetBeans como IDE.

Aclarado esto entonces, vamos a descargar Java SDK del siguiente sitio: https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

Segun el sistema operativo que estemos utilizando ya sea windows o linux vamos a escoger la version de la arquitectura de nuestro procesador que es de 64 bits. En el caso de windows descargamos el instalador el .exe mientras que para linux utilizamos el .tar.gz



Descarga e instalación en Linux Mint


Para empezar cabe aclarar que linux mint 19.02 ya cuenta con una version de java, pero nosotros instalaremos la version 8 para correr nuestra aplicación.



Entonces, para hacer esto podemos borrar completamente la version de java que viene incluida o podemos instalar nuestra version manualmente y seleccionarla como nuestra version por defecto.

Cuando descarguemos nuestro paquete Java SDK, abriremos una terminal en la carpeta de descargas.

Ahora descomprimimos el archivo y lo movemos a la carpeta "/usr/lib/jvm" con el siguiente comando:


"sudo tar xfz jdk-8u241-linux-x64.tar.gz -C /usr/lib/jvm/"



Ahora accederemos a la carpeta /usr/lib/jvm/jdk1.8.0_241 que es la que se creo al descomprimir.

Y entonces cambiamos la configuracion de java por la nueva versión.



Despues de cambiar las variables del entorno vamos a agregar la nueva version a las alternativas de java



Al final podremos seleccionar la version de java con la que queremos trabajar y entonces habremos terminado la instalación en linux



A partir de este punto ya podemos compilar archivos .java y generar nuestras aplicaciones. Solo que para nuestro caso, la aplicacion que probaremos es un proyecto de Netbeans, asi que tambien lo descargaremos e instalaremos.



En el caso de linux, el archivo descargado sera un .sh asi que tendremos que ejecutarlo en la consola.

Para eso abrimos una terminal en la carpeta de descargas y añadimos permisos para ejecutar el instalador.

"sudo chmod +x netbeans-8.0.1-javaee-linux.sh"

Despues ejecutamos el instalador con "sudo ./nebeans-8.0.1-javaee-linux.sh"



El instalador no tiene complicación, simplemente aceptamos los terminos y condiciones y seleccionamos el lugar donde se instalará.



El siguiente paso es muy importante puesto que debemos seleccionar la version del SDK que acabamos de instalar. La cual ya acomodamos por defecto, pero en caso de no estar seleccionada simplemente la buscamos entre las opciones del combo.



Despues simplemente comenzará a instalarse.



Al final nos quedara nuestro IDE listo para usar con java SDK



Y ahora ejecutamos nuestra aplicación, la cual esta conectada con mysql a nuestra base de datos que nos permite almacenar usuarios de diferente tipo, asi como las ventas del dia, los productos, proveedores, clientes, etc.



Basicamente, cada una de las tablas y sus relaciones tienen funcion de CRUD para nuestro negocio.



Cada formulario esta vinculado con una tabla distinta. Ademas de usar procedemientos almacenados para actualizar los inventarios cada que se realiza una compra.



Aquí podemos ver la base de datos.



Descarga e instalación en Windows


En el caso de windows es mucho mas facil el procedimiento. Lo primero que debemos hacer es descargar el SDK de java junto con netbeans. En el caso de Linux ya vimos como realizar la instalación por separado. Pero en el caso de windows podemos optar por descargar e instalar las dos cosas al mismo tiempo a través de un solo paquete.



Ya descargado el instalador, lo ejecutamos para proceder.



Luego nos preguntara donde instalar el sdk y el IDE



Al final comenzará la instalación



Por ultimo ejecutamos la aplicación cuando el instalador termine. Y entonces terminamos de probar en windows.
